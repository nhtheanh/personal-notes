## 1. Add

### 1.1 Add tất cả thay đổi

```bash
git add .
```

### 1.2 Add một file nào đó

```bash
git add /projects/file1.txt
```

### 1.3 Add một folder nào đó

```bash
git add /projects
```

## 2. Commit

```bash
git commit -m "Tóm tắt nội dung thay đổi"
```

## 3. Push

```bash
git push origin master
```
